# Push and PR — SmartScanner

Push the current branch and open a PR. Run this after `update-docs-and-commit` completes.

Pre-conditions:
- Working tree must be clean
- Branch must NOT be `master`
- Branch name must follow `feat/phase-NN-<short-name>` or `fix/<component>-<issue>`
